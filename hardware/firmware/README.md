# Firmware – Sensuto Ring

Versión 0.1 (desarrollo inicial)

## Plataforma

- Framework: PlatformIO / Arduino
- Sensores: MAX30102 (PPG), BMI270 (IMU)
- Energía: LTC3108 + TEG + PVDF
