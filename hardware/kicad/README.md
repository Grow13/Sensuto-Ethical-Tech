# Proyecto KiCad – Sensuto Ring

Este directorio contiene los archivos del diseño electrónico del Anillo Sensuto.

## Archivos
- `sensuto_ring.kicad_sch` – Esquemático
- `sensuto_ring.kicad_pro` – Proyecto
- `sensuto_ring.kicad_pcb` – PCB (próximamente)
- `/lib` – Librerías personalizadas

## Abrir el proyecto
1. Instalar KiCad 6 o 7.
2. Abrir `sensuto_ring.kicad_pro`.
