# Sensuto Ring – Hardware Design

Esta carpeta contiene los diseños de hardware del Anillo Sensuto, un dispositivo portable ético y de baja potencia para facilitar experiencias de conexión auténtica (sensuto).

## Estructura

- `/kicad` – Proyecto KiCad (esquemático, footprints personalizados)
- `/bom` – Lista de materiales (Bill of Materials)
- `/firmware` – Código fuente del microcontrolador
- `/mechanical` – Diseño 3D de carcasa y banda
